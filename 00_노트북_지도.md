| 노트북 | 역할 | 출력 |
|--------|------|------|
| 01_video_download | 영상 다운로드 (코랩+로컬 통합, 슬롯 5분할) | mp4 zip |

> 통합 이력: 구 09·10·11 → `04_video_pipeline` / 구 05(로컬 다운로드) → `01`에 흡수 / 유틸 2개 → `02` 부록으로 흡수.

## ③ 모델 실험 — `3_modeling/` (실험 기록, 대부분 유지)

### `1_pipeline/` — 핵심 흐름 (베이스라인→실험→최종)
| 노트북 | 실험 | 결론 |
|--------|------|------|
| 04_baseline | 베이스라인 XGB/CB | 기준점 확보 |
| 06_x_interval_experiment | X구간 pitch10~20 등 | ✅ **pitch15 확정** |
| 07_delta_experiment | delta feature | ✅ 채택(p=0.017) |
| 10_tuning_experiment | Optuna 3모델 | ✅ XGB 채택(+0.006) |
| 11_feature_selection | 피처 제거 실험 | ✅ 전체 59개 유지 |
| **13_final_evaluation** | 최종평가+SHAP+**E9 오차분석** | 🏆 **채택 모델 확정** |

### `2_experiments/` — 실험/기각 ("정직한 검증" 소재)
| 노트북 | 실험 | 결론 |
|--------|------|------|
| 05_nan_experiment | 결측 처리 5종 | ❌ baseline 유지 |
| 09_outlier_experiment | 이상치 처리 | ❌ baseline 유지 |
| 12_biomechanical_experiment | 영상 융합 + **E8 제구타겟 + subgroup** | ❌ 전면 기각(전방위, 핵심소재) |
| 14_domain_feature_experiment | A추세 + B변동성 | ❌ 기각(효과없음) |
| 15_sequence_model_experiment | 1D-CNN 시퀀스 | ❌ 기각(t=-20, p≈0) |
| 16_changepoint_experiment | 변화점 feature | ❌ 기각(diff-0.0001, p=0.70) ✅결론셀 추가됨 |

### `3_analysis/` — 해석·타겟·분류
| 노트북 | 실험 | 결론 |
|--------|------|------|
| 08_shap_analysis | 절대값 vs delta SHAP | delta 하위권 확인 |
| 18_target_comparison | Y 4종 × 영상융합 비교 | whiff만 예측가능 (✅전체데이터 표기 갱신) |
| 19_condition_classification | 좋은날/나쁜날 분류 + **E10 유형별AUC** | 🎯 AUC 0.65 (✅전체데이터 갱신) |
